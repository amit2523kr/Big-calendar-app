# React Big Calendar with Bar Graph

A React application that integrates React Big Calendar with Recharts bar graphs to visualize date-based data. Users can click on calendar dates to view detailed data in a modal with interactive bar charts.

## Features

- Interactive calendar with month, week, and day views
- Bar graph visualization for date-specific data
- Date highlighting for dates with available data
- Modal popup with detailed information
- Responsive design
- Redux state management
- Warning messages for dates without data

## Technologies Used

- React 18
- Redux Toolkit
- React Big Calendar
- Recharts
- Moment.js
- CSS3
